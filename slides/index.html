Consul

Service discovery and configuration made easy

Distributed Computing Denver

What is it...

A single binary executable that provides multi datacenter aware service discovery, failure detection, and key/value storage.

What it gives us...

  • Distributed key value store
  • Service discovery via DNS resolution
  • Monitoring and notification system for processes

Sounds Like?

Deployment

  • Install the Consul executable on every machine in your cluster
  • Provide that executable a single configuration file in json format
  • All communication with Consul is done through the locally running process
